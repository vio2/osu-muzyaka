# requirements

- OS: Windows
- osu!stable
- Python

# how to launch

1. clone or download this repo
`git clone git@github.com:vio2/osu-muzyaka.git`
2. get into working dir
`cd osu-muzyaka`
3. launch it with python
`python main.py`

if you need to change osu dir then change it

then just place replay into $osu_path/Replays dir or download some in game
